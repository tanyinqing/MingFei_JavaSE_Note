# Chapter 5 HTML 5

> HTML 4.01 - 1999

> HTML 5 - 2014

- Semantic Elements
    1.  header
        - 定义文档的页眉
    2.  nav
        - 定义导航链接
    3.  section
        - 定义文档中的节
        - 适用场景
            - 章节
            - 页眉 / 页脚 / 文档中的其他部分
    4.  aside
        - 定义其所处内容之外的内容
        - aside 的内容应该与附近的内容相关
    5.  article
        - 规定独立的自包含内容
        - 适用场景
            - 论坛帖子
            - 报纸文章
            - 博客条目
            - 用户评论
    6.  footer
        - 定义文档或节的页脚
